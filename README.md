## 계산기를 만들어보자!!
**STEP 1** (클래스 사용 없이)
계산기는 2개의 숫자를 받을 수 있고 사칙연산 될 문자를 받을 수 있다.
-> Scanner 객체를 생성해 사용자 입력을 받는다.

계산기는 exit을 입력할 때까지 계속해서 값을 받고 연산 결과를 반환한다.
-> 반복문 + 조건문을 활용해 종결조건 만족시 종료하는 로직 구현.

**STEP 2** (클래스 활용)
계산된 결과 값들을 기록하는 컬렉션을 만든다.
-> ArrayList 활용하여 계산 결과값들을 저장(history).

컬렉션의 가장 먼저 저장된 데이터를 삭제하는 기능을 만든다.
-> 위의 ArrayList를 통해 저장된 값들 중 가장 먼저 저장된 데이터 (인덱스가 0인)를 삭제하는 메서드 구현 (removeResult).

클래스를 활용하여 클래스별로 메서드를 정리한다.
-> Calculator 클래스는 게산만, App은 입력받는(시작) 기능만.

**STEP 3**
양의 정수만 받을 수 있었지만, 이제부터는 실수도 받을 수 있게 수정한다.
-> 제네릭 메서드 선언으로 실수(double)타입도 입력 가능하도록 구현.

결과가 저장되어 있는 컬렉션을 조회하는 기능을 만든다. 
-> 게터 메서드를 활용해 결과값 저장되어있는 컬렉션을 조회할 수 있도록 함 (getHistory).
